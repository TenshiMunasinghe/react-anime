#[Anifinder](https://tenshi-react-anime-project.netlify.com/)

Simple Anime finder that lets you filter by year and season providing links to official website and twitter.

####Technologies used :

- React
- hooks
- react-router-dom
- scss
- ShangriLa Anime API
